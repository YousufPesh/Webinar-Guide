Create an IT Help Desk Agent to Streamline Support and Ticket Creation

Help users quickly create and track IT support tickets, reducing wait times and improving the help desk experience for everyone.

AGENT CREATION
Create an Agentic AI Agent
Follow these steps to configure your IT Help Desk Agent

Step 1: Agent Name

Copy the agent name below and paste it into the 'Name this Agent' field.

IT Help Desk Agent

Step 2: Description

Copy the description below and paste it into the 'Description' field.

This Agent Helps Students with IT Help Desk Requests.

Step 3: Instructions

Copy the instructions below and paste them into the 'Instructions' field.

You are a helpful, kind assistant who helps create a ticket(incident) in ServiceNow. If the user says, "Could you please create a ticket on service now?", you should ask for a description and generate a short description for this ticket. Confirm the generated short description and description with the user. After the user's confirmation, call the new_tool action and pass these two inputs. If the user provides their own description, use it exactly as given and generate a short description. Confirm the generated short description and description with the user. After user's confirmation, call the new_tool action call the n8n servicenow action and pass these two inputs. After calling the new_tool action, you should receive the ticket link as well as the ticket(incident) number. Provide the successful message, ticket link, and ticket number to the user. If you do not receive the details about the ticket link and incident number, this means an error.

Step 4: Conversation Starters

Copy the conversation starter below and paste it into the 'Conversation starters' field.

I need help with my computer.

Step 5: Category

Set the category to 'Information Technology Service' (should be pre-selected).

Information Technology Service

Step 6: Set Access Permissions

To control who can access this agent:

  1. Scroll down to the Access Permissions section
  2. You can then:
    • Choose Only Me to keep the agent private
    • Or choose Groups to share the agent with selected teams

To share with groups:

  • Select one or more from the Group Information table
  • Click Attach to add them
  • They'll show up in the Attached Group List

You can also detach groups later using Remove Attached Group.

This gives you control over visibility and access within your organization.

Step 7: Click Continue

Click Continue to configure your agent's action (this opens the Add Action modal).

Step 8: Create an Action

Follow these steps in the Add Action modal:

  • Click Add Action to open the Add Action modal
  • Provide a name for your action (e.g., create_ticket)
  • Click Format JSON to validate
  • Click Create to create the action

Step 9: Action Name

Copy the action name below and paste it into the action name field in the modal.

ServiceNow Ticket Creation

Step 10: OpenAPI JSON Configuration

Copy the OpenAPI JSON configuration below and paste it into the schema field in the modal.

{
  "openapi": "3.0.0",
  "info": {
    "title": "ServiceNow Ticket Creation",
    "version": "1.0.0",
    "description": "Create tickets in ServiceNow via n8n workflow"
  },
  "servers": [
    {
      "url": "https://your-n8n-instance.com/webhook/service-now-ticket",
      "description": "n8n webhook endpoint"
    }
  ],
  "paths": {
    "/create-ticket": {
      "post": {
        "summary": "Create a new ServiceNow ticket",
        "requestBody": {
          "required": true,
          "content": {
            "application/json": {
              "schema": {
                "type": "object",
                "properties": {
                  "short_description": {
                    "type": "string",
                    "description": "Short description of the ticket"
                  },
                  "description": {
                    "type": "string",
                    "description": "Detailed description of the issue"
                  }
                },
                "required": ["short_description", "description"]
              }
            }
        }
      }
  },
  "components": {}
}
